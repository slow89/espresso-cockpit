# Skin Calls the LLM Directly for Shot Analysis

Shot Analysis is generated by the Skin calling an operator-configured LLM provider directly from the browser, authenticated with an operator-supplied API key stored in Skin-local persisted settings. Provider access goes through the Vercel AI SDK (`ai` + `@ai-sdk/anthropic` + `@ai-sdk/openai-compatible`): Anthropic is the default (`claude-opus-4-8` unless the operator picks a model), and any OpenAI-compatible endpoint (Ollama, OpenRouter, Groq, …) can be configured instead with a base URL and model. Because the request originates in the browser, the practical constraint on "any provider" is CORS: only providers that accept direct browser requests work (Anthropic requires the `anthropic-dangerous-direct-browser-access` opt-in header, which we send).

Because a ~100-character API key is not typable on the tablet (and the gateway serves the skin over plain HTTP, so the clipboard API is unavailable there), the settings are provisioned at deploy time: `pnpm deploy:tablet` bundles `SHOT_ANALYSIS_*` values from the laptop's `.env` into the skin as `shot-analysis-provision.json`, and the skin applies that payload on boot — once per distinct payload, so manual edits on the tablet survive reboots while a redeploy with a rotated key wins again. The settings panel remains as a fallback and for inspection. The trade-off: the key ships inside the skin bundle and is fetchable as a static file by anyone on the LAN who can reach the gateway — acceptable for a home network with a spend-capped key.

This deviates from the Visualizer precedent, where third-party integrations live in the Bridge as plugins, for the same reason the Compatibility Reconnect Adapter exists: the Bridge is slow to change, and analysis is a Skin presentation concern that touches no machine, device, or readiness policy. A Bridge-side LLM plugin or a hosted proxy service remain possible later migrations; the cost of this choice is that the API key lives on the tablet and each tablet must be configured separately.
