# Skin Calls the LLM Directly for Shot Analysis

Shot Analysis is generated by the Skin calling the Anthropic API directly from the browser (`claude-opus-4-8`, browser access enabled), authenticated with an operator-supplied API key stored in Skin-local persisted settings. This deviates from the Visualizer precedent, where third-party integrations live in the Bridge as plugins, for the same reason the Compatibility Reconnect Adapter exists: the Bridge is slow to change, and analysis is a Skin presentation concern that touches no machine, device, or readiness policy. A Bridge-side LLM plugin or a hosted proxy service remain possible later migrations; the cost of this choice is that the API key lives on the tablet and each tablet must be configured separately.
